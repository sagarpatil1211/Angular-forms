1. Template driven form

Mostly work done in the component template. i.e form valilate in html page

Template driven forms is created using directives in the template. It is mainly used for creating a simple form application.

To enable template driven forms, first we need to import FormsModule in app.module.ts.

          <form #userName="ngForm" (ngSubmit)="onClickSubmit(userName.value)">
          <input type="text" name="username"  ngModel>
          <input type="submit" value="submit">
          </form>

Here, we used ngModel attribute in input text field.
